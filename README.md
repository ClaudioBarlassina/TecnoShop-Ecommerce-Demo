## 🇺🇸 English

Full Stack eCommerce Demo is a complete eCommerce platform built with a decoupled architecture between frontend and backend. The frontend communicates with a REST API developed using Express.

The system is designed with reusable, scalable, and fully responsive components.

---

### 🚀 Features

- Decoupled Frontend & Backend architecture
- REST API built with Node.js and Express
- Reusable and scalable components
- Fully responsive design
- Product filtering by category, brand, and price
- Real-time product search
- Automatic pagination (10 products per page)
- Persistent shopping cart
- Custom payment simulation
- Email notifications with Nodemailer
- Global state management with Zustand

---

### 🛠️ Tech Stack

**Frontend:**  
React · Vite · Zustand · CSS  

**Backend:**  
Node.js · Express · Nodemailer
